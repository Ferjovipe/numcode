# Dictionaries

The full frequency dictionaries are hosted on HuggingFace:

https://huggingface.co/datasets/FERJOVIPE/numcode-dictionaries

## Available Languages

| Language | File | Tokens |
|----------|------|--------|
| Spanish | GOLD_ES.txt | ~7M |
| English | GOLD_EN.txt | ~9.6M |
| Arabic | GOLD_AR.txt | ~4.8M |
| Chinese | GOLD_ZH.txt | ~2.27M |
| Portuguese | GOLD_PT.txt | ~1.77M |
| French | GOLD_FR.txt | ~1.71M |

## Format

Each line: ID TOKEN COUNT

Sorted by frequency (most common first).
